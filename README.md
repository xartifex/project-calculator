Задания и рекомендации на изучение EJB

Цель заданий:

Изучить технологию Enterprise JavaBeans (EJB 2.0 и EJB 3.0). Познакомиться с основными типами бинов (Entity Bean, Session Beans и Message Driven Beans). Научиться создавать и деплоить бины на сервер. Используя систему логирования сообщений, отследить их жизненый цикл.

Задание 1:

Написать web-приложение, выполняющее функции калькулятора. Приложение должно выполнять все основные операции вычисления на сервере в бине. Калькулятор должен уметь считать как десятичные числа, так и шестнадцатиричные. Сохранять информацию о выполняемых операциях (аргументы, тип операции и результат) в базе данных, в логах и отображать историю выполняемых операций на странице приложения.

Рекомендации:

Web-приложение должно работать на EJB 2.0 и на EJB 3.0. Пользователю предоставляется возможность самому выбирать версию используемых бинов.

Для операций с десятичными числами следует использовать Stateless Session Bean и выполнять операцию вызовом одного метода на бине, который будет принимать параметры и возвращать результат. Для операций с шестнадцатиричными числами следует применять Stateful Session Bean и выполнять операцию вызовом двух методов на бине: первый метод будет принимать параметры и производить вычисления, а второй возвратит результат, полученный после выполнения первого метода.

Для сохранения информации в базу при выполнении операций с десятичными числами следует использовать Container Managed Persistence Entity Bean, а для операций с шестнадцатиричными числами – Bean Managed Persistence Entity Bean. В EJB 3.0 используются POJO классы технологии JPA для обоих систем счисления.

Сохранение информации в лог следует производить с использованием Message Driven Bean и системы логирования сообщений. Информация пишется в лог не напрямую, а через бин, который асинхронно пишет в лог принимаемое сообщение. Для десятичной системы исчисления применяются бины, подписанные на Queue, а для шестнадцатиричной – бины, подписанные на Topic.

В web-приложении необходимо применять AJAX технологию, используя популярные фреймворки: jQuery или Prototype.

Задание 2:

Написать web-приложение, позволяющее посмотреть на работу транзакционных атрибутов в Container Managed Persistence Entity Bean. Рассматриваются случаи работы бина в нутри транзакции и без транзакции. Результаты работы атрибутов отображать в логах.

Рекомендации:

Web-приложение должно работать на EJB 2.0 и на EJB 3.0. Пользователю предоставляется возможность самому выбирать версию используемых бинов.
